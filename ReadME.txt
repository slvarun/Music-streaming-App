1. Create a new Folder
2. Create a virtual environment inside this folder
3. Get my music streaming application folder into your newly created folder
4. Install the required libraries present in requirements.txt
5. After this first execute db.py file only which is present inside the code folder
6. After executing , a music.db file will be created in the code folder with all the required tables
7. After this you can execute app.py file and view my music streaming application
